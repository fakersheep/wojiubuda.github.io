我有只仓鼠，叫小综，品种老公公，母的。

十分难搞定。

你去抓它，它跑的旋风快，徒手超难抓；你好不容易抓它在手，它必会在0.5秒内翻身掉出你的的手心，多一刻不停留；你稍微握紧它，只露出它激萌的小脸欣赏，它见挣脱无望，看着你，略略略，把颊囊里的小谷粒吐出来，“哎呀怎么吐了”，你赶紧放它回笼子。

说什么都不跟你亲近，1秒都不行，0.5秒都不行。

周一工作准备

wojiubuda 2017-07-23

本篇目的：工作准备，中吉综合机的逻辑、前端、后台、手机端，咖啡机的逻辑、维护端。

中吉综合机的功能点

PC后台：
  1. 给机器的每个货道配货、换货、合并货道。货道很难拆下来，怎么合并
售货机端：
  1. 同步商品信息：商品编号、名称、图片、价格、数量。永久化信息到本地。
  2. 售卖：选货道进入商品详情
    • 进入详情前：判断货道是否可售（可售定义如下)，如果可售，则查询所选货道的商品信息。
      • 货道无故障 能否出货前查询所选货道是否故障
      • 货道有库存
      • 货道无连续2次出货失败记录
    • 收到订单已支付消息后，提示支付成功。
    • 出货中：检测出货是否成功，展示状态（出货中，出货成功/出货失败，请联系客服）。
    • 出货后：
      • 出货成功：更改该货道库存，缓存订单编号 为什么 上报出货成功状态，清除上次出货失败记录。
      • 出货失败：上报出货失败，存储出货故障。
  3. 故障查询与上报 是否有自动查询故障的机制
  4. 维护模式
    • 纸币可以退回、硬币可以找零、不买货可以找零。不用显示在屏幕上。
    • 遥控出货开关
    • LED灯开关
    • 温度开关：1区2区是什么，设置温度有什么用，温度上限和下限有什么用
    • 制冷加热：（只要开制冷，自动开启玻璃除雾）
    • 钱币预装： 点了有什么用
    • 清除故障：有什么用
    • 测试货道出货
  5. 广告、取货码

售卖模式原型地址：https://modao.cc/app/vex81uF5N9SS4sHR0cSBuE6FMQS1p4V

wojiubuda 2017-07-23

阿爸今年26了。
阿爸想要变得厉害一些。

厉害起来了，可能有2个好处。

  • 生孩子前后，都有地方抢着要阿爸。
  • 或者，阿爸可以靠自己来过得钱，而不是公司的一部分。

阿爸怎么能“厉害起来”呢？

  1. 工作需要迅速反馈吧，所有能做的事情按分钟计，往前赶，不断推进，不断完善，一往无前。
  2. 搞明白自己，然后兼济天下。怎么兼济天下呢？就是自己用的着的干货，总结下来，分享给尽可能多的人。分享自己的微笑号二维码。
  3. 不知道为啥，用还惦记着python……我只能说实在无聊的话，健身学跳舞做水膜泡脚喝热水吃水果这些有意义的休闲我都不考虑的话，实在的实在无事可做，不要看电视，要抄敲python。没错，抄敲。

一些工作岗位，运营、产品、总助，是没办法明确衡量一个人的价值的。
在这个意识和表象的世界，你需要用一件一件事情，对一方面一方面的人，怼自己的口碑。插自己的flag。flag插满公司，这样，以后的工作也好，事情也好，机会也好，都会有人主动找你。

香水

wojiubuda 2017-07-21

缓缓吸气，感受空气的讯息，让好闻的味道和自己融为一体。

昨天看了一部电影《香水》。

这部分是剧透，介意勿看。

变态男主一生对味道痴迷，他创造的香水配方风靡巴黎，让门可罗雀的香水铺子变得人满为患。

变态男主成年后发现，最好闻的味道，是年轻貌美的姑娘的味道，想永久保存。于是潜心学会味道提取术，谋杀了多个姑娘，最终制成了极其好闻的香水。

有多好闻呢？ 男主被判死刑，行刑前，滴几滴香水在手绢上，挥向看行刑的人群，众人波浪般倾倒，如痴如醉。然后都给他跪下，为什么呢，太好闻了，不可能是坏人。大家一致同意放走他。

变态男主就带着这小瓶王者香水，回到自己的故乡。香水给他力量，他可以用香水去得到很多东西。但是似乎不能得到爱？总之他杀都没干，直接来到他出生的地方，把那罐香水，全部倒在自己身上。然后，人们僵尸一样冲向他，抱着他，贴着他，亲他，啃食他……直到啃个干净，什么都没剩，人群散去……天亮了……

我对味道，也比较在意。嗅到好的气味可以给人极大的满足感和幸福感。

香蕉 橙子 西瓜 玫瑰 崽崽

所以，看到男主轻闭着眼睛，缓缓深深庄严吸气的时候，略有同感。

但是不能理解的是，喜欢貌美姑娘的味道，好好追到手，结个婚，每天闻活的不好么? 以他创造香水配方的能力，多赚些银子哄来个姑娘应该不是问题。何必把姑娘某杀掉，再萃取姑娘的味道呢……

即使他成功萃取了姑娘们的味道，有机会永久保存这样的味道，那又怎样，没多久香水又被他以“一口气倒在自己身上”的方式霍霍没了。 也没有保存多久啊。

摊手，精神病人的世界，好像缺根弦儿，不懂。

还有就是，这个世界上也许有一些，我们无法理解的人。所以，作为女孩子，对这个世界怀有未知的敬意，不要单独走夜路，不要到人少的地方，不要考验人性。

阿弥陀佛。

售货机的逻辑

wojiubuda 2017-07-19

写此篇的目的

我们要卖中吉的综合机，软件需要我们自己写。所以我们要根据中吉给的底层协议，来实现售货机的功能。

售货机需要哪些功能？

共性功能：

  1. 初始化：导入初始相关数据，使其可售卖。
    • 条件：[售货机端请求]时
    • 内容：每个货道对应的商品信息：图片、中英文名称、价格、库存。
  2. 同步更新：后台更新的数据，同步到售货机端。
    • 条件：[机器重启]时
    • 内容：商品图片、中英文名称、价格、库存。
  3. 售卖：展示支付二维码前查询可售状态、收到支付成功的推送后出货、收到出货结果后上报且更新可售状态。
    • 展示支付二维码前查询可售状态
      • 条件：用户输入货道号时
      • 查询内容
        • 货道是否故障
        • 商品是否售空
        • 是否连续2次出货失败历史
    • 收到支付成功的推送后 出货
      • 展示状态：出货中、出货成功、出货失败。
    • 收到出货结果后
      • 出货成功：更改该货道库存，上报出货成功状态，清楚上次出货失败记录。
      • 出货失败：上报出货失败状态，存储此次出货失败记录。
  4. 故障查询：定时、特定触发条件查询机器故障。
    • 查询内容：暂无货道故障以外的故障，需要电话问。
  5. 故障上报：定时、或特定触发条件上报机器故障。
  6. 心跳：监控网络状态。

个性功能

  1. 性能测试
    • 调货检测灵敏度
  2. 设置
    • 制冷加热系统：制冷 加热 关闭
    • 设置温度上下限
    • 玻璃除雾：开关
